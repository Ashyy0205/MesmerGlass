# MesmerGlass Quick Start (v1.0)

This is a fast “get visuals on screen” walkthrough for the v1.0 UI.

## 1) Launch MesmerGlass

```powershell
python -m mesmerglass run
```

## 2) Choose where visuals appear

1. Open the **Display** tab
2. Select the monitor(s) you want MesmerGlass to use
3. (Optional) If using wireless VR, connect your headset in **VR Devices (Wireless)**

## 3) Start a session

1. Open the **Home** tab
2. Create a new session (or open an existing one)
3. Confirm the session shows as loaded

## 4) Run a cuelist

1. Open the **Cuelists** tab
2. Create/select a cuelist
3. Start playback from the session runner controls

## 5) Optional: connect a device

1. Open the **Device** tab
2. Scan/connect via MesmerIntiface (built-in)

## If nothing shows up

- Confirm you selected at least one display in the **Display** tab.
- Check the bottom status bar for the selected display summary.
- If you’re on a multi-monitor setup, try selecting only your primary display first.
